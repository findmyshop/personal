Sharing this Site

If you would like to share this site with family or friends, please provide them with the original user name that was supplied to you so that they can create a unique user name and explore the program in their own way.

They can set up their own user name and explore the content that they are most interested in. They will not be able to see your questions or answers, but you may find it helpful to print out and share your answers with each other while you are discussing these issues.
