Supported Browsers

For the optimal experience, we recommend using an updated browser such as Chrome, Firefox or IE9 or higher. 

You can download these browsers for free by clicking on these links: Firefox - Chrome 

Users with earlier versions of IE may experience some issues due to their browser’s functionality. If you have IE8 installed on your machine, we suggest that you upgrade to Chrome or Firefox and access this program exclusively with the updated browser.
